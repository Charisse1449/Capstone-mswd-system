MSWD Complete Super Admin UI Prototype

How to run:
1. Extract this ZIP.
2. Open the folder in VS Code.
3. Open Terminal.
4. Run: npm install
5. Run: npm start

Login:
Username: admin
Password: admin123

What is included:
- Complete Super Admin frontend UI
- Dashboard statistics and notifications
- Residents list, add resident, edit/deactivate, resident profile, fingerprint registration demo
- Beneficiaries management demo
- Assistance Programs management demo
- Claims workflow demo with fingerprint verification and blockchain hash generation
- Blockchain ledger and integrity verification demo
- Reports with PDF/Excel placeholder buttons
- Master Data: Categories and Barangays
- Dynamic dropdowns: added categories/barangays appear in Add Resident form
- System Users
- Roles & Permissions with permission matrix
- Audit Logs
- Settings: General, Database, Fingerprint Scanner, Blockchain, Backup & Restore, Notifications

Note:
This version uses browser localStorage as a frontend demo database. It does not require MySQL yet. Data saves inside the app while using the same browser/Electron storage.
